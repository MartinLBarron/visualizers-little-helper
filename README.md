## Installation

1. Clone Repo

```
git clone https://github.com/MartinLBarron/visualizers-little-helper.git
```

2. Copy contents of repo to ~/.claude/skills/
```
cp -r visualizers-little-helper ~/.claude/skills/
```

or within Claude

```
/plugin add /path/to/skill-directory

```
